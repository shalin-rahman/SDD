## Locale tags (BCP 47)

| Today | Gap |
|-------|-----|
| `en`, `fr`, `bn` language-only tags | Need **`en-US`**, **`bn-BD`**; keep `fr` as `fr-FR` optional Wave 3 |
| `document.documentElement.lang = locale` (2-letter) | Must emit full tag; persist BCP 47 in storage |
| Mobile `Locale('en')` / `languageCode` only | Must use `Locale.fromSubtags(languageCode: 'en', countryCode: 'US')` |
| Org profile `locale` field accepts `en`/`fr`/`bn` | Align with BCP 47 default `en-US` |
